Creating a Report (Infor SCS)

Technical Writing Sample by Avisek Dalal

Home Portfolio

Purpose

This document describes the process of creating and saving a custom report in Infor SCS (Sales & Catering Solution).

Steps

  1. Access the Reports Module
    Go to Reports → Create Report.
  2. Select a Report Type
    Choose a base report from the list or create a new one. Click Next.
  3. Set Filters and Parameters
    Define filtering criteria such as Date Range, Property, Sales Representative, or Event Type.
  4. Add Columns
    Choose data fields to include in the report output. Arrange them in the desired order.
  5. Preview and Validate
    Click Preview to review the data output. Make any adjustments needed.
  6. Save the Report
    Enter a name and optional description. Click Save to store the report.

Outcome

The report is successfully created and can be accessed later under Reports → My Reports for viewing, editing, or scheduling.
