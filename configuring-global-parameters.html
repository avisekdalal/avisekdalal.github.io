Configuring Global Parameters (Infor Factory Track)

Technical Writing Sample by Avisek Dalal

Home Portfolio

Purpose

This procedure explains how to configure parameters that apply to all Infor Factory Track CSI transactions and forms.

Steps

  1. Open the Global Parameters Form
    Navigate to Configuration → Parameters → Global Parameters.
    The Global Parameters form is displayed.
  2. Specify Configuration Options
    • Kanban Implemented: Select this check box to enable Kanban functionality.
      • Note: If not selected, Kanban processes are not supported in Factory Track.
    • Kanban Label: Specify a label ID for Kanban labels.
      • Note: This field appears only if the “Kanban Implemented” parameter is selected.
    • Handling Units in Use: Select this check box to enable handling units.
    • ERP Lot is Supplier Lot: Select this check box to automatically use the supplier’s lot number as the ERP lot number.
    • EAN Code in Use: Select this check box to allow users to scan European Article Numbers (EAN) instead of item numbers.
      When an EAN is scanned, the corresponding item number is displayed if a match is found.
    • ICS Code in Use: Select this check box to allow scanning of Item Code System (ICS) numbers instead of item numbers.
      When an ICS number is scanned, the related item number is displayed if a match is found.
    • Module: Specify the module to which this parameter applies: WM, TT, SF, or All.
    • Shop Floor in Use: Select this check box to enable access to the Shop Floor module.
    • TT Implemented: Select this check box to enable Time Track–related transactions in Factory Track.
    • None: Select this check box if neither Shop Floor nor Time Track is implemented.
    • PIN Required: Select this check box to make a PIN mandatory when employees sign in as a generic user.
    • Restrict Movement for Pick List Serials: Select this check box to restrict consumption of serials already picked against a pick list.
  3. Save the Configuration
    Click Save to apply the global parameters.

Notes

  • Only users with administrative privileges can modify these parameters.
  • Review configuration changes carefully to avoid conflicts between modules (Shop Floor, Time Track, Warehouse Management).
  • Document parameter settings for audit and rollback purposes.

Outcome

After completing this configuration, Factory Track applies the defined parameters globally across all modules and transactions.
