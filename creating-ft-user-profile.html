Creating a Factory Track User Profile (Infor Factory Track)

Technical Writing Sample by Avisek Dalal

Home Portfolio

Purpose

This guide explains how to create and configure a new user profile in Infor Factory Track to manage user access and security.

Steps

  1. Open the User Profiles Form
    Navigate to Administration → User Management → User Profiles.
    The User Profiles form appears.
  2. Create a New User Profile
    Click New and enter the required details:
    • User ID: Enter a unique identifier for the user.
    • User Name: Enter the user's full name.
    • Password: Specify a temporary password (the user can reset it later).
    • Employee Number: Link the user to the corresponding employee record.
    • Default Warehouse: Assign the warehouse where the user primarily operates.
  3. Assign Roles and Permissions
    In the Roles tab, select one or more predefined roles that determine access rights.
    Optionally, fine-tune permissions under the Access Rights tab to restrict or expand access to specific functions.
  4. Set Preferences
    Configure default form views, themes, and regional settings according to the user’s location and role.
  5. Save the Profile
    Click Save to create the user profile.

Outcome

The new user profile is created and ready for use. The assigned permissions and roles determine the user’s access within Factory Track.
