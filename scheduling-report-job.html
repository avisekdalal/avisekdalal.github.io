Scheduling a Report Job (Infor SCS)

Technical Writing Sample by Avisek Dalal

Home Portfolio

Purpose

This procedure explains how to schedule report jobs to run automatically in Infor SCS.

Steps

  1. Access the Report Scheduler
    Navigate to Reports → Schedule Report Job.
  2. Select a Report
    Choose the report you want to schedule from the list.
  3. Set Frequency
    Define how often the report should run — Daily, Weekly, Monthly, or Custom.
  4. Set Output Options
    Choose the file format (PDF, Excel, CSV) and specify the destination (email or folder).
  5. Activate Schedule
    Click Save and Activate to enable the schedule.

Outcome

The report job is scheduled to run automatically at defined intervals, reducing manual effort and ensuring timely data delivery.
