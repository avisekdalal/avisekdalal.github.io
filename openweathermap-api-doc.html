OpenWeatherMap API Documentation

This API provides access to current weather data, forecasts, and historical weather for any location worldwide.

Base URL

https://api.openweathermap.org/data/2.5/

Authentication

All requests require an API key.

appid=YOUR_API_KEY

üå§Ô∏è Current Weather Data

GET /weather

Retrieve current weather by city, lat/lon, or zip.

Example:

GET https://api.openweathermap.org/data/2.5/weather?q=Toronto&appid=YOUR_API_KEY&units=metric

Sample Response:

{
  "weather": [{"main": "Clouds", "description": "broken clouds"}],
  "main": { "temp": 22.5, "humidity": 63 },
  "wind": { "speed": 3.6 },
  "name": "Toronto"
}
